Overview

This project focused on the process of drop forging steel components and overmolding a polymer handle to create a functional screwdriver.

Process

  • Heating and drop-forging tool steel blank
  • Machining the tip and shank to tolerance
  • Injection molding the plastic handle

Materials

O1 tool steel for the shaft and Phillips tip, ABS plastic for handle molding.
