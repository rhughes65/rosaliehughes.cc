Overview

This project demonstrates a mechanical binary counter that advances manually. The counter converts rotational motion into binary output using cams, linkages, and pawls.

Features

  • 6-bit mechanical binary display
  • Hand-cranked motion stepper
  • Spring-loaded detents for each binary state

Learning Outcome

This project combined mechanical design principles with binary logic, demonstrating how digital systems can be represented physically.
